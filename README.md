# 🛍️ Cosabella — Relojes & Accesorios

Aplicación web e-commerce desarrollada para la gestión y visualización de una tienda de relojes y accesorios, con enfoque en una experiencia de usuario moderna, elegante y similar a plataformas como Amazon y Mercado Libre.

---

## 🚀 Descripción del proyecto

Cosabella es una tienda digital que permite:

* Visualizar productos por categorías
* Navegar por un catálogo estructurado
* Ver detalles de productos
* Gestionar productos desde un panel administrativo
* Simular experiencia de compra moderna
* Integración visual con WhatsApp para pedidos

El diseño está enfocado en una estética **premium (negro + dorado)** para transmitir elegancia y exclusividad.

---

## 🧠 Tecnologías utilizadas

### Backend

* Python 3.12
* Flask
* Jinja2 (templates)

### Frontend

* HTML5
* CSS3 (estilos personalizados en `main.css`)
* JavaScript (interacciones básicas)

### Base de datos

* MySQL (principal)
* MongoDB (evaluado para expansión)

### Herramientas

* Visual Studio Code
* Git & GitHub
* XAMPP (entorno local)

---

## 🎨 Diseño y experiencia de usuario

El frontend fue desarrollado con inspiración en grandes plataformas de e-commerce:

* Amazon
* Mercado Libre
* Éxito

Se implementaron:

* Sistema de tarjetas de productos (product cards)
* Grid responsive para catálogo
* Sidebar de filtros
* Navegación clara y estructurada
* Botones de acción visibles (Catálogo / WhatsApp)
